Documentation
API
Constructor
Popup([base], layout, contents, openTrigger, [closeTrigger], [cssStyle], [ms])
Creates a Popup with the specified properties. closeTrigger, cssStyle, and ms are optional. ms specifies how many seconds the popup should persist, and is only applicable when closeTrigger is "time".
Properties
base
Required only for "important" popups (i.e., those that hide the background contents). This is the main body element of the HTML file.
layout
Required. The layout of the Popup. Choices: "text", "box", "important".
contents
Optional. String containing the contents of the Popup.
openTrigger
Required. What makes the Popup open? Choices: "click", "mouseover", any other valid event type
closeTrigger
Optional. What makes the Popup close? Choices: "time" (automatically closes after specified time), "close button" (closes when close button is clicked), any valid event type
cssStyle
Optional. String containing the name of a CSS style.
ms
Required only when closeTrigger is "time". Specifies how many milliseconds the Popup should persist for before disappearing.
animationName
Optional. String containing the name of the animation for this Popup.
animationDuration
Optional. The duration of the animation for this Popup.
animationIterations
Optional. The number of iterations for the animation for this Popup.
animationDelay
Optional. The delay for the animation for this Popup.
soundPath
Optional. The path to the sound file that plays when this Popup is triggered.
parentElement
Optional. The HTML element to append the Popup to. By default, the Popup is a child of the parent of the triggering element.
collateral
Optional. An array of HTML elements which should be animated when this Popup is triggered.
collateralAnimationName
Optional. String containing the name of the animation for this Popup's collateral elements.
collateralAnimationDuration
Optional. The number of iterations for the animation for this Popup's collateral elements.
collateralAnimationIterations
Optional. The number of iterations for the animation for this Popup's collateral elements.
collateralAnimationDelay
Optional. The delay for the animation for this Popup's collateral elements.
Methods
addOpenTrigger(element)
Set which HTML element should trigger the Popup when an event (openTrigger) occurs.
setParentElement(parent)
Set which HTML element should be the parent element of the Popup element.
addAnimation(name, duration, iterations, delay)
Set the animation name, duration, iterations, and delay for the Popup.
addSound(path)
Set the path to the sound file that should play when the Popup is triggered.
setCollateral(arrayOfCollateralElements)
Set the array of HTML elements which should be animated when the Popup is triggered.
setCollateralAnimation(name, duration, iterations, delay)
Set the animation name, duration, iterations, and delay for the Popup's collateral elements.
